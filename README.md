# Sales Analytics Dashboard

## Project Overview

This project presents an interactive Sales Analytics Dashboard built using Power BI, SQL, and Excel. The dashboard provides insights into sales performance, profitability, customer behavior, and business trends.

## Key Features

* Total Sales Analysis
* Profit Analysis
* Customer Analytics
* Order Tracking
* Profit Margin Calculation
* Average Order Value Analysis
* Regional Sales Performance
* Customer Segmentation
* Monthly Revenue Trend
* Top Customer Identification

## Tools Used

* Power BI
* SQL (MySQL)
* Microsoft Excel
* GitHub

## Dashboard KPIs

* Total Sales: 2.30M
* Total Profit: 286.40K
* Total Orders: 5K
* Total Customers: 793
* Profit Margin: 12.47%
* Average Order Value: 458.61
